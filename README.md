# Time Series Energy Forecasting (ERCOT - Electric Reliability Council of Texas)

This project builds an end-to-end time series forecasting pipeline for electricity demand using ERCOT (Texas) data.

## Project Structure

- Notebook 01: Data ingestion and validation
- Notebook 02: Exploratory Data Analysis (EDA)
- Notebook 03: Forecasting models (coming soon)

## Tech Stack

- Python
- Pandas
- Matplotlib
- scikit-learn
- statsmodels

## Data Source

U.S. Energy Information Administration (EIA) API  
https://api.eia.gov/

## Notes

Data is retrieved via API and not stored in this repository due to size restrictions.
Run Notebook 01 to fetch and generate datasets locally.
